# Water Availability Awareness

> *"Water is the driving force of all nature."* — Leonardo da Vinci

A static awareness website built as part of an academic project at **Lovely Professional University (LPU)**, focused on educating users about global and Indian water scarcity — its causes, social/economic impacts, solutions, and latest news.

---

## Live Preview

Open `index.html` in any modern browser to run locally.

---

## Pages

| File | Description |
|---|---|
| `index.html` | Main page — landing, about, causes, solutions, news |
| `aboutus.html` | Team / project information |
| `contactus.html` | Contact form |

---

## Sections (index.html)

- **Landing** — Full-screen hero with quote
- **About** — Overview of water availability + India water scarcity map
- **Causes** — Climate change, pollution, overuse & mismanagement (with linked PDFs)
- **Social & Economic Impact** — Effects on communities and livelihoods
- **Solutions** — Education, rainwater harvesting, smart irrigation
- **News** — Bangalore, Maharashtra, and Rajasthan water crisis coverage

## Tech Stack

- **HTML5** — Structure and content
- **CSS3** — Styling (`styles.css`)
- **JavaScript** — Interactions (`script.js`)
- **Google Fonts** — Poppins font family
- No frameworks, no build tools — pure static site

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/ryoikitenkaai/Water-awareness.git
   cd Water-awareness
   ```

2. Open `index.html` in your browser:
   ```bash
   # On Linux/Mac
   open index.html

   # Or just double-click the file in your file explorer
   ```

> No server or install required.
